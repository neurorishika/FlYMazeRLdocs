def FlYMazeRL.agents.classical.QLearner.trial_step (   self,
  state 
) 
Perform a single Q-Learning step in the environment.
- Make a choice based on the current state.
- Get reward and new state.
- Update according to the equation:
    Q[state, action] = (1 - learning_rate) * Q[state, action] + learning_rate * (reward + discount_rate * max(Q[new_state,:]))
==============================================================================================================================

Parameters:
state: current state (int)

Returns:
new_state: new state (int)
done: whether the episode is done (bool)
